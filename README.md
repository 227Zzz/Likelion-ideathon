## 📢 About Project
최근 1인 가구 및 고령층 증가로 인해 사회적 문제로 대두되는 '고독사'를 조기에 예방하고자 기획된 서비스입니다. 사용자의 생활 패턴을 AI가 분석하여 위험도를 측정하고, 능동적으로 환경을 제어하거나 보호자 및 지자체와 연계하여 위급 상황에 대처합니다.

- **타겟 유저**: 혼자 거주하는 고령층, 따로 거주하는 노부모를 둔 자녀, 지자체 복지 담당자

## ⭐ Key Features
1. **행동 패턴 이상 감지**
   - 움직임, 식사, 수면 패턴 데이터를 수집하여 평소 패턴과 다를 경우 보호자에게 알림 전송.
2. **AI 능동형 환경 제어**
   - 사용자 패턴 및 상태에 따라 강제로 조명 켜기, 커튼 열기, 음악 재생 등 주변 환경을 능동적으로 변화시킴.
3. **음성 기반 감정 분석 서비스**
   - AI와의 대화를 통해 사용자의 대화 패턴과 목소리의 높낮이 등을 분석하여 우울 정도를 측정.
4. **지자체 및 응급 기관 연계**
   - 지역 내 고위험군 데이터를 분석하여 지자체에 제공하고, 3단계(위험 상태) 돌입 시 보호자 알림 및 응급 기관 자동 연결.
